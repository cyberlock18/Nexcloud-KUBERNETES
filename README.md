# Nexcloud-KUBERNETES
Bienvenido a mi TFG de ASIR, en este proyecto generé mediante dos servidores una vlan interna basada en contenedores.
Realicé una VPN privada para poder acceder remotamente desde cualquier parte del mundo a mi entorno.
Este entorno tenía varias zonas de trabajo en donde dependiendo del usuario y sus roles permitia acceder a un sistema de ficheros compartidos o modificarlos.
Los coontenedores y pods se crearon para trabajar sobre código y poder compilarlo y el administrador podia levantar instancias para visualizar las métricas de los contenedores y sistemas.
